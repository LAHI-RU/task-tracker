# Task Tracker (Mini MERN)

A small but real MERN project to practice:
- Node.js + Express REST API
- MongoDB (Mongoose)
- React (Vite)
- Tailwind CSS v4
- CRUD + validation + error handling

## Features
### Backend
- Create / Read all / Read one / Update / Delete tasks
- Validation with meaningful messages
- Consistent API responses
- CORS configured
- Environment variables (.env)
- Simple logging

### Frontend
- List tasks
- Add task
- Change status (todo/doing/done)
- Delete task
- Loading + error states
- Minimal Tailwind UI

## Tech Stack
- Backend: Node.js, Express, Mongoose
- Frontend: React (Vite), Tailwind CSS v4
- Database: MongoDB
